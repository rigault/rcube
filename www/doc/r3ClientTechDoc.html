1. Introduction

Le client utilise les techniques de développements Web classiques. HTML, CSS et Javascript

L'essentiel est rédigé en Javascript.

Le client consomme l'API Json exposée par le serveur.

Il utilise pour cela les instructions fetch Javascript.

Un plugin aisgpsserver écrit en langage C permettant de retrouver les informations AIS et GPS est décrit dans l'annexe consacrée au serveur.

Le client utilise l'API windy pour l'affichage du fond de carte météo.

L'API Windy repose sur l'API leaflet.

L'API sweetalert2 est utilisée pour la gestion des alertes des boites de dialogues, des tableaux.

L'API plotly est utilisée pour la représentation des graphiques.

2. Documentation

La documentation est constituée :

  • Des fichiers html, css que vous lisez et qui sont dans le répertoire www,
  • de la documentation produite par la commande jsdoc *.js
